# 압구정 분양가능 평형·조망각 추천 앱

기존 랭크앱에서 순위 산출과 분양가능 평형 예측만 가져오고, 조망 시뮬레이터의 계산 결과를 연결한 새 Streamlit 앱입니다.
구글시트 인증은 사용하지 않고 앱 폴더 안의 엑셀/CSV 파일을 직접 읽습니다.

## 실행

```bash
streamlit run app.py
```

## 공시가격 데이터 넣는 위치

아래 둘 중 하나의 파일명으로 공시가격 시트를 저장하세요.

```text
data/main_data.xlsx
data/main_data.csv
```

앱은 헤더 행에서 `구역`, `단지명`, `동`, `호` 컬럼을 자동으로 찾아 읽습니다.
엑셀 파일은 `공동주택 공시가격` 탭을 읽고, 순위 기준은 정식 `2026` 컬럼입니다.
`2026 예비` 컬럼은 순위 계산에 사용하지 않습니다.

## 2026 순위 보정

2026년 공시가격 동률은 순위 계산용으로만 보정합니다.

- 최고층 바로 아래층을 최고 기준으로 둡니다.
  예: 13층 건물은 12층, 12층 건물은 11층.
- 1층부터 최고 기준층까지는 1층 가격과 최고 기준층 가격 사이를 균등 배분합니다.
- 꼭대기층은 최고 기준층보다 한 층 아래 가격과 같게 둡니다.
  예: 13층은 11층 가격, 12층은 10층 가격.
- 원본 `2026` 값은 수정하지 않고, 앱 내부의 순위기준가격으로만 사용합니다.

## 조망 추천 기준

현재 선택 호수의 층은 미래 조망 계산에 사용하지 않습니다.
내 순위로 안전하게 진입 가능한 `예상 가능 평형`을 먼저 대상으로 조망각을 추천합니다.
예상 가능 평형이 조망 데이터에 없으면 조망 데이터에 실제 입력된 바로 아래 평형으로 맞춰 표시합니다. 예를 들어 4구역 `68평`은 조망 데이터에 `68평`이 없으면 `67평` 기준으로 조망을 계산합니다.
참고 후보는 기준 평형보다 낮거나 같은 평형대 중 `최고 한강조망각`이 기준 평형보다 더 큰 경우에만 `조망 우위 참고 후보`로 표시합니다. 조망각이 기준 평형보다 떨어지는 하위 평형은 제외합니다.
4구역은 지도상 타입 표기를 기준으로 평형을 매칭합니다. 예: `96 Type=40평`, `105 Type=43평`, `114/115 Type=47평`, `120 Type=49평`, `126 Type=52평`, `147 Type=60평`, `154 Type=63평`, `164 Type=67평`, `166 Type=68평`, `177 Type=72평`, `183 Type=74평`, `188 Type=77평`, `P230=94평`, `P270=110평`, `P420=172평`.
미래 배치도 유닛의 층별 조망각을 계산하되, 같은 `한강조망각`이 이어지는 층은 `56∼65층`처럼 한 구간으로 묶어 표시합니다.
같은 평형이 여러 동 또는 여러 유닛에 걸쳐 있으면 각 동·타입·층구간별 조망각을 따로 보여주고, 같은 가능평형 안에서 해당 구간이 차지하는 층수 비중을 `추첨확률`로 표시합니다.
조망표는 평형별로 먼저 묶어 표시하며, 예를 들어 `87py 조망 후보` 아래에 87py의 동·층구간·확률만 모아 보여준 뒤 다음 평형을 표시합니다.
분양가능 평형에 `P` 표시가 있는 펜트하우스가 포함되면, 조망 GeoJSON에 별도 유닛이 없더라도 일반층 조망표보다 먼저 `펜트하우스 우선 후보`로 표시합니다.
각 평형별 조망표 하단에는 `10도 이하`, `10∼20도`, `20∼30도` 같은 조망각 구간별 추첨확률 리포트를 표시합니다.

2구역 배치도 원본에서 102동 `65`로 들어가 있던 유닛은 70평형으로 보정했습니다.
106동 65평형 유닛 색상은 `#F5CBCB`로 맞췄습니다.

## 2구역 복수소유자 합산

2구역에서 지정된 2주택 소유자는 순위 산정 시 원래의 개별 순위를 제외하고 합산 1명으로 계산합니다.
화면에는 `다주택 소유자 합산`과 두 주택의 합계 공시가격·합계 순위기준가격을 표시합니다.
분양가능 평형 계산에서는 복수소유 8쌍을 반영해 2구역 조합원 수를 1,916명, 일반분양을 342세대로 봅니다.
별도의 복수소유 1% 추정보정은 적용하지 않습니다.

## 포함 데이터

- `data/view_summary.json`: 조망 추천용 유닛별·층별 요약 데이터
- `data/view_unit_rows.csv`: 유닛 기준 요약 CSV
- `data/view_floor_rows.csv`: 층별 조망각 CSV

현재 앱은 기존 동호수의 구역 내 순위로 분양가능 평형을 계산하고, 미래 배치도 GeoJSON에서 그 평형에 해당하는 유닛을 찾아 조망각이 큰 순서로 추천합니다.
